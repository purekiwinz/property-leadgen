Exporting as 1080 × 1080 px images

In Chrome: right-click an ad → Inspect → select the .ad element → click ⋮ (three-dot menu in Elements panel) → Capture node screenshot.
Set browser zoom to 200% first to export at true 1080 × 1080 px.

To add suburb aerial photos: set background-image: url('photo.jpg') on the .ad element and add background-size: cover. The existing overlay keeps text legible.
